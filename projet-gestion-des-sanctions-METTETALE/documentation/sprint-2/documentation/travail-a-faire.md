## 📁 Structure du Projet

Le projet suit une architecture MVC (Model-View-Controller) avec :
- **Controllers** : Logique métier et gestion des requêtes
- **Models/Repositories** : Accès aux données et logique métier
- **Views/Templates** : Interface utilisateur et présentation
- **Routing** : Système de routage personnalisé
- **HTTP** : Composants pour la gestion des requêtes/réponses
